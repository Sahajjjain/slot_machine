# 🎰 Python Slot Machine

A simple command-line Slot Machine game built using Python.

## Features

* Deposit money before playing
* Choose the number of betting lines
* Place bets on each line
* Random slot machine spins
* Win money when symbols match across a line
* Balance tracking system
* Input validation for a smooth user experience

## Technologies Used

* Python 3
* Random Module

## How It Works

1. Deposit an amount.
2. Select the number of lines to bet on (1–3).
3. Enter your bet amount per line.
4. Spin the slot machine.
5. Matching symbols on a line result in winnings.
6. Continue playing until you choose to quit.
